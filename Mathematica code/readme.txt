This folder contains the Mathematica code for the MOCHA model.

23_09_07_miniMOCHA-theory-plots_3.nb: run the model and put together the model simulations shown in Figure 2 of the manuscript.

23_09_08_model_fits_4.nb: fit the model to the data, and plot the outcome.

23_09_07_3-structure-miniMOCHA-model_1.m: define model and optimal allocation rules.

23_09_07_fitting_1.m: functions for data fitting

23_09_07_load_data_1.m, 23_09_07_load_bacteria_1.m: load experimental data

23_09_07_plot_data_and_simulations_4.wl: functions for plotting

23_09_08_export_parameters_1.nb, 23_10_30_export_data_points.nb: export fitted parameters and simulations
